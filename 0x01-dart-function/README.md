# Dart Function Practice

Un parcours pour maîtriser les fonctions en Dart : paramètres, retours, structures de contrôle, et manipulation de collections.

## Ressources recommandées
- Dart - Syntax
- Dart - Conditions
- Dart - Loops
- Dart - Map
- Dart - Cheatsheet
- Iterables Collections

## Objectifs d'apprentissage
1. Définir et invoquer des fonctions Dart avec différents types de retour et paramètres.
2. Intégrer de la logique conditionnelle et des boucles au sein des fonctions pour contrôler l'exécution.
3. Appeler des fonctions depuis d'autres fonctions et illustrer la récursivité lorsque c'est pertinent.
4. Manipuler des chaînes, listes et maps à l'aide des méthodes natives de Dart dans le corps des fonctions.

## Mise en pratique
- Implémente des fonctions simples dans `0x01-dart-function` pour illustrer chaque objectif ci-dessus.
- Assure-toi que chaque fichier Dart se termine par un saut de ligne et peut être compilé avec Dart 3.x.x sur Ubuntu 20.04.

## Contraintes
- Tous les fichiers doivent être compatibles avec Dart 3.x.x sur Ubuntu 20.04 LTS.
- Chaque fichier Dart doit se terminer par une nouvelle ligne.
- Ce README doit rester à la racine du dossier de projet.
