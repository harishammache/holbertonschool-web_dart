# holbertonschool-web_dart

## Description

Ce projet marque le début de votre aventure dans le développement mobile avec Dart, un langage moderne développé par Google. Vous apprendrez les bases du langage Dart, sa syntaxe claire, son typage fort, et comment il est utilisé pour créer des applications performantes, notamment avec Flutter.

## Objectifs pédagogiques

À la fin de ce projet, vous serez capable de :
- Comprendre la structure et la syntaxe de base de Dart
- Déclarer et utiliser des variables avec les types courants (`int`, `double`, `String`)
- Afficher des sorties dans la console
- Utiliser les instructions de contrôle (`if`, `else`)
- Comprendre le rôle de la fonction `main()` comme point d'entrée d'un programme Dart

## Organisation du dépôt

Exemple de structure pour les exercices :

```
0x00-hello-dart/
├── 0-hello_holberton.dart
└── ...
```

## Installation

1. Installer Dart : suivez la documentation officielle — https://dart.dev/get-dart

2. Vérifier l'installation :

```bash
dart --version
```

3. Cloner le dépôt :

```bash
git clone <url-du-repo>
cd holbertonschool-web_dart
```

## Exécution

Pour lancer l'exemple de base :

```bash
dart 0x00-hello-dart/0-hello_holberton.dart
```

Exemple attendu :

```
Hello Holberton!
```
